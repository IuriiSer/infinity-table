# Getting Started

- `npm i` to init project
- `npm start` to start the react app

# Main Idea

Идея проекта - реализовать простую оптимизацию для большого набора данных. Реализованны следующие механизмы
- lazy loading для данных через IntersectionObserver, пакетами по 100 элементов
- оптимизация рендера: происходит memo компонента, что рендерит пакет данных
- оптимизация памяти: происходит очистка устаревших пакетов
